安装hexo

在安装hexo之前要先安装node.js和Git。

安装完Git后打开Gitbase程序，然后创建博客的根目录：

安装hexo，创建完博客的根目录后cd d:/myblog 接下来只需要使用 npm 即可完成 Hexo 的安装，

安装完hexo后，初始化myblog文件目录，

初始完化，在myblog文件夹下会出现如下的文件，

  • node_modules:是hexo主要依赖的插件
  • public：是hexo 生成的静态文件存放的文件夹，同时也是发布到github文件夹
  • scaffolds：存放相关模板的文件夹
  • source：存放文章、标签、分类等的文件夹
  • themes：存放主题的文件
  • _config.yml：博客全局配置文件

选择Scheme

Scheme 是 NexT 提供的一种特性，借助于 Scheme，NexT 为你提供多种不同的外观。同时，几乎所有的配置都可以 在 Scheme 之间共用。目前 NexT 支持三种 Scheme，他们是：

  • Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白

  • Mist - Muse 的紧凑版本，整洁有序的单栏外观

  • Pisces - 双栏 Scheme，小家碧玉似的清新
