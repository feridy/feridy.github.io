利用hexo + next + github搭建这人博客

发表于 2018-08-05 | 分类于 经验 ， blog

本篇文章我主要将我如何搭建这个博客的经验进行分享，我将从头到尾来进行详细的介绍如何利用HEXO+next+github来搭建博客，我在搭建过程中也遇到了一些问题，我也将进行讲解。

安装hexo

在安装hexo之前要先安装node.js和Git。

安装完Git后打开Gitbase程序，然后创建博客的根目录：

安装hexo，创建完博客的根目录后cd d:/myblog 接下来只需要使用 npm 即可完成 Hexo 的安装，

安装完hexo后，初始化myblog文件目录，

初始完化，在myblog文件夹下会出现如下的文件，

  • node_modules:是hexo主要依赖的插件
  • public：是hexo 生成的静态文件存放的文件夹，同时也是发布到github文件夹
  • scaffolds：存放相关模板的文件夹
  • source：存放文章、标签、分类等的文件夹
  • themes：存放主题的文件
  • _config.yml：博客全局配置文件

初始化完成之后，输入命令启动hexo，来查看一下博客是否搭建完成

然后在浏览器上输入http://localhost:4000/

配置github用来发布你的博客

如果没有github账户就新建一个账号，然后新建一个repository，repository库的名字形式为yourname.github.io的格式，yourname为你github的名字

  • 配置git中的用户名和邮箱 yourUserName和yourUserEmail替换成你自己的github的用户名和邮箱

  • 检查.ssh的文件是否存在
